# The Pinnacle – Web Page 🏙️

A responsive landing page built for **The Pinnacle by STJ Group**, a luxury residential project located in Mohali, Punjab.

🔗 **Live Demo:** [https://dpnxhuu.github.io/The-Pinnacle-Web-Page/]
---

## About The Project

This is a real estate landing page for **The Pinnacle Mohali** — a G+35 luxury apartment tower developed by STJ Group under the Mohali City Centre (MCC) umbrella.

The page showcases:
- Project overview and highlights
- Floor plans, pricing, and amenities
- Gallery, location, and contact section
- Downloadable brochure option
